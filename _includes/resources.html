Docker Image and Tools for Cloud Implementations

Wallace is available as a Docker image on Docker Hub (currently ver 1.1). Anyone who has Docker installed on their system can use this image to run Wallace without the need to interface with R. Follow these steps:

  1. Check that Docker is running on your system
  2. At a command line, run the following command: 'docker pull mlammens/wallace_v1:PE.1.1'
  3. Next, run the following command: 'docker run -d -p 8080:8080 mlammens/wallace_v1:PE.1.1'
  4. Open a browser and navigate to 'localhost:8080'

Wallace can also run in the CyVerse Visual Interactive Computing Environment (VICE). We have developed a VICE App that is currently available by contacting Matt Aiello-Lammens at maiellolammens@pace.edu, and will be publically available soon.
